# ✨ Overview

**Cafe17** is a modern digital menu website designed for **cafes and restaurants**.

The goal of the application is to:

• present menu items in a clean interface  
• improve the customer browsing experience  
• enable simple digital ordering  

The system combines several key components:

* Product Menu
* Smart Filtering
* Shopping Cart
* WhatsApp Order Integration

This creates a **smooth and interactive menu experience** for customers.

---

# 🚀 Features

### 🍽 Product Menu System

Products are organized into clear menu categories.

Each product includes:

* name
* description
* price
* image
* detail modal

---

### 🔍 Search & Filtering

Users can easily explore the menu with:

* product search
* category filtering
* sorting system

---

### 🛒 Shopping Cart

Customers can create a quick order using the cart.

Features include:

* add to cart
* remove items
* update quantity
* order summary

---

### 🌍 Language & Currency Switch

Supports multiple languages and currencies:

* Turkish / English language switch
* EUR / TRY currency switch

---

### 📱 Responsive Design

The website adapts to all screen sizes.

Supported devices:

* mobile
* tablet
* desktop

---

### 📦 WhatsApp Ordering

Customers can send orders directly through **WhatsApp**.

The system automatically generates an order message.
